##＃ 前端項目項目配置(Vite)

  1. 使用vite| npm init vite@latest
  2. 安裝nvm | brew install nvm
  3. 安裝node| brew install node
  4. Vite選擇Vue和typeScript
  5. 安裝package.json依賴| npm instal
  6. 啟動項目| sudo npm run dev

##＃ 前端項目配置(Vue)

  1. 安裝Vue腳手架| npm init vue@latest ->一路yes到底
  2. npm install
